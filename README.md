# мiцеlium Translator

Современный переводчик мiцеlium на React + Vite.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте браузер по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

### Предварительный просмотр сборки

```bash
npm run preview
```

## ✨ Особенности

- ⚛️ **React 18** - современная библиотека для UI
- ⚡ **Vite** - быстрая сборка и горячая перезагрузка
- 🎨 **Темная тема** - переключение между светлой и темной темой
- 📚 **Вкладки браузера** - переключение между переводчиком и словарем
- 🔍 **Поиск** - быстрый поиск по словарю
- 🧪 **Экспериментальный режим** - альтернативная цветовая схема
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- 🎭 **Модальные окна** - красивые окна для просмотра переводов
- 💾 **LocalStorage** - сохранение настроек пользователя

## 🎮 Использование

### Переводчик

1. Введите текст в поле ввода
2. Нажмите "Зашифровать" для перевода в мiцеlium
3. Нажмите "Расшифровать" для обратного перевода
4. Используйте кнопку "КОПИРОВАТЬ" для копирования результата

### Словарь

1. Переключитесь на вкладку "Словарь"
2. Выберите "Алфавит" или "Словарные слова"
3. Используйте поиск для быстрого нахождения
4. Кликните на элемент для просмотра деталей
5. Копируйте перевод из модального окна

## 🎨 Настройки

- **🌙/☀️** - Переключение темной/светлой темы
- **🧿/🪼** - Переключение акцентного цвета
- **🧪** - Экспериментальный режим (красная цветовая схема)
